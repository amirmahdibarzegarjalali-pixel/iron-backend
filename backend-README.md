# بک‌اند پروتکل آیرون (نسخه‌ی تک‌فایلی)

این نسخه عمداً در یک فایل (`server.js`) نوشته شده — بدون هیچ پوشه‌ی تودرتو —
تا از موبایل هم بشه بدون مشکل روی گیت‌هاب آپلودش کرد. عملکردش دقیقاً همون
نسخه‌ی قبلیه (حساب کاربری، ورود با ایمیل/موبایل، کد تایید دومرحله‌ای، پروفایل،
آپلود عکس، رکوردها، اندازه‌گیری بدن).

## فایل‌هایی که باید آپلود کنی
فقط همین ۴ فایل، هیچ پوشه‌ای در کار نیست:
- `server.js`
- `package.json`
- `.env.example`
- `README.md` (اختیاری)

## آپلود از موبایل روی گیت‌هاب
1. برو توی ریپازیتوری‌ت روی گیت‌هاب
2. اگه از قبل فایل/پوشه‌ای اونجا هست، همه رو پاک کن (برای شروع تمیز) — یا یه ریپازیتوری جدید بساز
3. **Add file → Upload files**
4. این ۴ فایل رو (تک‌تک یا با هم، چون هیچکدوم پوشه نیستن) انتخاب کن و آپلود کن
5. **Commit changes**

## روی Render
1. **Settings → Root Directory** رو کامل خالی بذار (چون دیگه پوشه‌ی iron-backend‌ای در کار نیست)
2. **Start Command:** `node server.js`
3. **Build Command:** خالی
4. Environment Variables: `NODE_ENV=production`، `JWT_ACCESS_SECRET`، `JWT_REFRESH_SECRET` (رشته‌های رندوم)، `CORS_ORIGIN` (آدرس Netlify‌ت)
5. اگه پلن Starter گرفتی و می‌خوای داده‌ها پایدار بمونن: `STORAGE_DIR=/var/data` + Add Disk با Mount Path `/var/data`
6. **Manual Deploy → Deploy latest commit**

بعد از دیپلوی موفق، `/health` باید `{"ok":true}` برگردونه.
